myCBR is a modular tool implemented in Java that is based on a SDK that has been extended to a workbench for defining case representations, modelling similarity measures and rapid prototyping with case bases. Further, the SDK was extended with a Rest API allowing integration with other programming languages (such as Python).

SDK

  • myCBR SDK repository: https://github.com/ntnu-ai-lab/mycbr-sdk
  • myCBR SDK binary: https://folk.idi.ntnu.no/kerstinb/mycbr/sdk/myCBR-3.3-SNAPSHOT.jar

Extensions of the SDK: Rest API

  • myCBR Rest API: https://github.com/ntnu-ai-lab/mycbr-rest
  • Examples of using the Rest API: https://github.com/ntnu-ai-lab/mycbr-sample-python
